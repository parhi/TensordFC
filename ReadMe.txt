This project consists of files to extract spatial components, temporal component,
subjecwise components (A, B, C) from a stacked sliding window correlation matrix.

1. B. Sen and K. K. Parhi, "Predicting Biological Gender and Intelligence from fMRI via Dynamic Functional Connectivity," 
IEEE Transactions on Biomedical Engineering, doi: 10.1109/TBME.2020.3011363.

Add all the folders and then run the scripts.
